===================================================
           MP3专辑封面下载器 v1.1.0
===================================================

【软件功能】
自动从QQ音乐、网易云音乐、酷狗音乐下载MP3专辑封面并嵌入到MP3文件中。
如专业音乐平台未找到封面，则使用必应图片搜索获取可能的专辑封面。
成功添加封面后可自动将文件重命名为 YYYYMMDD.原文件名.mp3 格式。
如文件已有封面，则只执行重命名操作（除非使用--no-rename选项）。
同时自动更新MP3文件的元数据（歌曲名、艺术家、专辑名）。

【网络要求】
程序使用国内音乐平台API，需要在中国大陆网络环境下运行。
境外网络会导致连接失败，请使用国内网络或手机热点。

【封面搜索来源】
搜索优先级: QQ音乐 > 网易云音乐 > 酷狗音乐 > 酷我音乐 > 必应图片搜索
（注: 酷我音乐API因平台升级CSRF验证暂不可用，其余三个平台均正常）

【使用方法】
1. 直接拖放MP3文件或文件夹到程序图标上
2. 通过命令行使用更多高级功能，可用参数如下:
   路径            - MP3文件或目录路径(默认为当前目录)
   --delay 秒数    - 设置每首歌曲处理间隔时间(秒)，默认0.5秒
   --no-rename     - 不重命名文件，仅添加封面
   --force-download - 即使已有封面也强制重新下载
   --verbose       - 显示详细的处理信息

【使用示例】
MP3专辑封面下载器.exe D:\音乐文件夹
MP3专辑封面下载器.exe 单个文件.mp3 --delay 1
MP3专辑封面下载器.exe --no-rename
MP3专辑封面下载器.exe --force-download --verbose

【从源码构建】
1. 安装依赖:  pip install -r requirements.txt
2. 运行程序:  python mp3_cover_downloader.py [参数]
3. 一键打包:  python build_exe.py --zip

构建脚本说明:
  python build_exe.py                # 仅构建 exe
  python build_exe.py --release      # 构建 + 生成发布目录
  python build_exe.py --zip          # 构建 + 生成 zip 发布包
  python build_exe.py --clean        # 仅清理构建产物

【依赖项】
- Python 3.10+
- requests >= 2.32.5
- mutagen >= 1.47.0
- PyInstaller >= 6.0（仅打包需要）
